操作方法（ゲーム時）

  ・「←↓↑→」
      それぞれ左、下、上、右に移動
  ・HJKLキー（Win32のみ）
      それぞれ左、下、上、右に移動
  ・カーソルキー（Win32のみ）
      移動

操作方法（ゲームオーバー時）

  ・「Retry」
      再度ゲームをプレイ
  ・カーソル・スペース・Enter・Ctrl・右クリック・TAB（Win32のみ）
      再度ゲームをプレイ

遊び方

  ・十字キーで進行方向を選択します。
  ・数字を食べると、その分だけ長くなります。
  ・どんどん長くなろう。
  ・壁か自分にぶつかるとゲームオーバー。
