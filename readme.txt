操作方法（ゲーム時）

  ・「HJKL」
      それぞれ左、下、上、右に移動
  ・HJKLキー（Win32のみ）
      それぞれ左、下、上、右に移動
  ・カーソルキー（Win32のみ）
      移動
  ・Escキー（Win32のみ）
      ゲームを終了

操作方法（ゲームオーバー時）

  ・「Retry」
      再度ゲームをプレイ
  ・「Exit」（ゲームオーバー時 && iOS以外）
      ゲームを終了
  ・Escキー（Win32のみ）
      ゲームを終了
  ・カーソル・スペース・Enter・Ctrl・右クリック・TAB（ゲームオーバー時 && Win32のみ）
      再度ゲームをプレイ

遊び方

  ・十字キーで進行方向を選択します。
  ・数字を食べると、その分だけ長くなります。
  ・どんどん長くなろう。
  ・壁か自分にぶつかるとゲームオーバー。
